Marketing Analytics Widget

This widget provides an interactive interface for querying and visualizing marketing analytics data.

Live Demo

Installation

To add this widget to your Google Site:

  1. Go to your Google Site in edit mode
  2. Add an "Embed" element where you want the widget
  3. Copy and paste the following code:

<!-- Replace YOUR_BACKEND_URL with your actual backend server URL -->
<div id="marketing-analytics-widget" data-backend-url="YOUR_BACKEND_URL"></div>
<script src="https://sabbadoo32.github.io/full-funnel/frontend/widget.js"></script>

Features

  • Natural language queries for marketing data
  • Real-time data visualization
  • Campaign performance analysis
  • Interactive chat interface
